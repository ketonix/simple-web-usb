Ketonix Web-USB Example

Ketonix Green - Web-USB example Measure Breath Ketones.

Enable the Chrome flag "Automatic detection of WebUSB-compatible devices".

  1. Enter "chrome://flags" in Chrome URL field and hit enter.
  2. Search for "usb" in flags
  3. Enable "Automatic detection of WebUSB-compatible devices"
  4. Relaunch Chrome to make the setting effective.
